開発フローの状態機械・レビュー内部構造・フック配置を可視化した公式ドキュメント。
ブラウザで開いて閲覧する（単一HTML・外部依存なし・オフライン閲覧可）。

## 更新ポリシー

- **v3-current**: 実装変更（PR マージ）に追随して更新する。実装との齟齬は不許可。

## 図の凡例

- 青: 生成/実装/試験（生成系エージェント）
- 橙: レビュー（3成分・round≦3）
- 紫（明）: 軽量ゲート（gate-runner）
- 紫（濃）: 重ゲート（consistency-checker）
- 緑: ユーザー承認
- 赤: 是正（FIX）
- 緑バッジ「FLAT」: フラット化モデル固有
